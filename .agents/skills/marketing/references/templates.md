# Marketing Templates

## Changelog Entry

```markdown
### Added

- **Feature Name** — one-sentence description of what it does and why it matters. [#issue](link)
```

Rules:
- Bold the feature name
- Em-dash (—) separates name from description
- Present tense ("adds", "supports"), not past
- Link to issue/PR when available
- Group under: Added, Changed, Fixed, Removed, Tests

---

## Twitter / X Post (≤ 280 chars)

```
[emoji] [Feature headline in ≤8 words]

[1-2 sentence benefit statement]

[hashtags: #VSCode #AI #DevTools]
```

Example:
```
🎯 Branch selection in Agent Board

Pick the base branch before launching AI sessions — worktrees land exactly where you want them.

#VSCode #AI #DevTools
```
